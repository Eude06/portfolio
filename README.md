# Responsive Minimalist Portfolio

A clean, modern and fully responsive single-page portfolio website designed to showcase my software development projects and technical expertise without unnecessary navigation overhead.

## 🚀 Live Demo
Once deployed, you can view the live site here: 
[https://https://eude06.github.io/portfolio/](https://https://eude06.github.io/portfolio/)

## 🛠️ Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript
* **Framework:** Bootstrap 5 (Local Minified)
* **Form Handling:** Web3Forms API (Serverless)

## 📦 Features
* **Responsive Grid Layout:** Features an asymmetric split layout (`col-md-5` and `col-md-7`) that sits beautifully side-by-side on desktop views and seamlessly stacks on mobile screens.
* **Custom Color Palette:** Styled with a polished dark blue primary hue, cool slate secondary details, and vibrant cyan accents for subtle visual interest.
* **Functional Contact Form:** Fully integrated serverless contact form that securely forwards user inquiries straight to my inbox.
* **Menu-free Architecture:** A minimalist structure tailored for direct, straightforward scannability.

## 🖥️ Local Setup
To run or edit this project locally:
1. Clone the repository: `git clone https://github.com/Eude06/portfolio.git`
2. Open the directory and launch `index.html` directly in any modern browser.
